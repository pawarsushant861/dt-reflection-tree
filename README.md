# Daily Reflection Tree

This project is a deterministic reflection system designed as part of the DT Fellowship assignment.

## 📌 Overview

The system guides users through an end-of-day reflection using a structured decision tree.

* No AI at runtime
* Fully deterministic
* Fixed options for all questions
* Based on psychological frameworks

## 🧠 Axes Covered

1. Locus (Victim vs Victor)
2. Orientation (Contribution vs Entitlement)
3. Radius (Self vs Others)

## ⚙️ How It Works

* The user starts at the root node
* Each question provides fixed options
* Based on answers, the system routes to the next node
* Signals track user tendencies
* A summary is generated at the end

## 🚀 Features

* Deterministic branching
* No hallucination risk
* Clear and structured reflection
* Simple and user-friendly design
